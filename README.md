# Smart Campus Management System

A full-featured web application to manage college campus operations.

## Features
- Role-based login (Admin / Faculty / Student)
- Student & faculty management
- Attendance tracking
- Results management
- Leave approval system
- Announcements module

## Tech Stack
PHP | MySQL | HTML | CSS | Bootstrap | JavaScript

## Setup Instructions
1. Clone the repo
2. Import the SQL file into phpMyAdmin
3. Update DB config in assets/config/
4. Run via XAMPP localhost
